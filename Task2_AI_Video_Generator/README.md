# Task 2: Automated AI Video Generation Pipeline

## Description
A Python-based AI video generation pipeline that converts text prompts into cinematic videos using AI video generation APIs.

## Technologies Used
- Python
- Hugging Face API
- Requests

## Features
- Text-to-video generation
- MP4 output creation
- Automated API integration

## Example Prompt
- "Slow-motion fireworks and gold glitter effects over 3D golden text"

## How to Run

pip install -r requirements.txt
python video_generator.py
