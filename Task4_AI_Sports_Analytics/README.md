# Task 4: AI Sports Analytics Assistant

## Description

An AI-powered sports analytics assistant built using Python, Pandas, and Gemini API. The application allows users to ask natural language questions about sports statistics and match data.

## Technologies Used

* Python
* Pandas
* Google Gemini API
* Python Dotenv

## Features

* Natural language querying
* Sports statistics analysis
* Match data processing
* AI-generated insights

## Example Questions

* Which team won the most matches?
* Which venue hosted the maximum matches?
* What is the win percentage of Chennai Super Kings?

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python assistant.py
```
