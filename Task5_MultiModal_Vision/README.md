# Task 5: MultiModal Vision Analysis

## Description

A web application built using Flask and Google Gemini Vision API that analyzes uploaded images and generates AI-powered descriptions.

## Technologies Used

* Python
* Flask
* Google Gemini API
* Pillow

## Features

* Image upload interface
* AI-based image understanding
* Object detection
* Scene description generation

## Note

The original task suggested using GPT-4o (Vision) or Claude 3.5 Sonnet with Node.js. For this implementation, Google Gemini 2.5 Flash Vision with Python and Flask was used as an equivalent multimodal AI solution. The application fulfills the same functional requirements, including image understanding, OCR, and visual analysis.

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
