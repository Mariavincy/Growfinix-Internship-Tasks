# Task 5: MultiModal Vision Analysis

## Description

A web application built using Flask and Google Gemini Vision API that analyzes uploaded images and generates AI-powered descriptions.

## Technologies Used

* Python
* Flask
* Google Gemini API
* Pillow

## Features

* Image upload interface
* AI-based image understanding
* Object detection
* Scene description generation

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
